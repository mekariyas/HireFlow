# HireFlow
A PERN job board platform that allows users to apply for jobs and companies to create jobs, view applicants, edit  and delete job posts.
It utilizes protect routes, file uploads for both user and companies.

## Tech Stack
 ### Front-end
 - React
 - Tailwind CSS
 - Typescript
 - Axios
 - React Query
 - React Hook Form
 - Zustand
### Backend
- Node js
- Express js
- Typescript
- Zod
- Multer
- Cloudinary SDK
- Jsonwebtokens
- Express rate limiter
- Neon
- Drizzle
## How to install
  ```
   git clone https://github.com/mekariyas/hireflow
  ```
  ### Frontend
  ```
    cd frontend
    npm install
    npm run dev 
  ```
  ### backend
  ```
     cd backend
     npm install
     npm run dev 
  ```
