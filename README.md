# HireFlow
A PERN job board platform that allows users to apply for jobs and companies to create jobs, view applicants, edit  and delte job posts.

## Tech Stack
 ### Front-end
 - React
 - Tailwind CSS
 - Typescript
 - Axios
 - React Query
 - React Hook Form
 - Zustand
### Backend
- Node js
- Express js
- Typescript
- Zod
- Jsonwebtokens
- Express rate limiter
- Neon
- Drizzle
## How to install
1. Clone the repository using git clone
2. cd Hireflow
### Frontend
1. cd frontend
2. npm install
3. npm run dev
### backend
1. cd backend
2. npm install
3. npm run dev
